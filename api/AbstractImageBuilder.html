AbstractImageBuilder

This AbstractImageBuilder class provides all the basic methods that an ImageBuilder could need and almost all the methods an ImageBuilder must implement.

Return the control widgets for [ QueryDataModel, PipelineModel, LookupTableManager ] for only those who were provided at the creation.

Return the following set of objects.
