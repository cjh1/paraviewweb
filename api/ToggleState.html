ToggleState

ToggleState represents a two value state with notification mechanism when its state change.

The class can be imported with the following command:

import ToggleState from 'paraviewweb/src/Common/State/ToggleState';

toggleState() #

Toggle the current state which will trigger a notification. That method is generated via a closure which allow it to be used as a callback anywhere without any binding issue with invalid this.

Register a callback to any possible change of the model. The callback method profile is as follow.
